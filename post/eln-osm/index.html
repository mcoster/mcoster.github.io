Electronic Lab Notebooks (ELN) for the Open Source Malaria (OSM) initiative: A video tutorial

Thursday, 27 July 2017 Electronic Lab Notebook, ELN, OSM, Open Source Malaria, tutorial

This video tutorial gives an overview of how to set up and create entries in an Electronic Lab Notebook (ELN). The focus is on how to use the LabTrove ELN system to contribute to the Open Source Malaria initiative as a synthetic chemist.
